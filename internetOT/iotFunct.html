¿Cómo Funciona?

Un sistema de IoT completo está integrado por cuatro componentes

  • Sensores/Dispositivos

    Estos se encargan de recolectar datos de su entorno y puede ser tan simple como una lectura de temperatura o algo más complejo como una transmisión de video.

    En realidad, los sensores pueden agruparse o pueden formar parte de un mismo dispositivo que tiene una variedad de funciones (por ejemplo, un celular, que tiene múltiples sensores, como la cámara, GPS, entre otros). Sin embargo, ya se que se trate de un sensor independiente o de un dispositivo compuesto por varios sensores, este se encarga del primer paso, que es recopilar datos del entorno

  • Conectividad

    Lo siguiente es lograr que los datos recopilados lleguen a la nube. Los sensores y dispositivos se pueden conectar a la nube a través de una variedad de métodos, por ejemplo, WiFi, Bluetooth, redes de áre amplia de baja potencia (LPWAN), o incluso pueden conectarse directamente a Internet a través de Ethernet. Cada una de estas opciones de conectividad tiene sus propias compensaciones en cuanto al consumo de energía, el rango y ancho de banda; elegir la mejor opción depende de cuál sea la aplicación específica del sistema de IoT, sin embargo, todas ellas permitirán llegar al objetivo: llevar los datos a la nube.

Dependiendo de la aplicación de IoT, el usuario puede realizar acciones y afectar el sistema, por ejemplo, puede ajustar de forma remota la temperatura de un almacenamiento en frío a través de una aplicación en su celular.

Algunas acciones las realiza automáticamente el sistema, a través de reglas predefinidas, como por ejemplo notificar a las autoridades que hay un intruso de manera directa en lugar de decirle al usuario [Ulo vigilance system].

  • Procesamiento de Datos

    Una vez que los datos llegan a la nube, es el software el que se encarga de procesarlos. El procesamiento de los datos depende, nuevamente, de la aplicación específica del sistema de IoT, por lo tanto el procesar los datos puede ser un proceso tan simple como verificar que la lectura de temperatura esté dentro de un rango aceptable, o mucho más complejo como usa visión por computadora en video para identificar objetos (o intrusos en tu casa). Después de esto el usuario es el que debe intervenir, lo que nos lleva a nuestro último componente.

  • Interfaz de Usuario

    La información debe llegar al usuario de una manera, esto podría ser por medio de un email, una notificación a su celular, un mensaje, etc. Por ello, el usuario requiere de una interfaz que le permita tener acceso al sistema de forma proactiva, por ejemplo, si el usuario quiere revisar las transmisiones de video en su casa requerirá entonces de una aplicación en su celular o una aplicación web.

En resumen: Un sistema de IoT consiste de sensores y dispositivos que “hablan” con el almacenamiento en la nube a través de algún tipo de conectividad, y una vez que los datos llegan a la nube, un software procesa toda esa información para decidir qué hacer con ella sin que el usuario intervenga, o bien, el usuario puede decidir qué hacer, siendo una aplicación web o móvil el medio para llevar a cabo las acciones deseadas y realizar ajustes si así lo desea.
