Seguridad en la Web

El propósito de la seguridad web es prevenir ataques dirigidos a la misma para evitar que los usuarios sean víctimas de los diversos peligros del internet como estafas financieras o robo de información personal. Así pues, la seguridad web es la acción/práctica de proteger sitios web del acceso, uso, modificación, destrucción o interrupción, no autorizados; esta protección requiere de un esfuerzo de diseño que va más allá de simplemente la aplicación web, sino que involucra configuración del lado del servidor, entre otros aspectos.

La ciberseguridad es importante porque los sitios web desprotegidos están expuestos a sufrir situaciones como:

  • Robo de información guardada en el servidor web y minería de criptomonedas.
  • Explotación de datos personales y realizar descargas de software malicioso.
  • Redireccionamiento a páginas web maliciosas y anuncios no deseados.

Peligros principales de la web

Malware

El malware es un software creado con fines maliciosos, diseñado para infectar y dañar un sistema. Abarca vulnerabilidades que van desde los virus informáticos hasta el adware que puede infectar tanto a las computadoras como a las páginas web. Dos de los tipos más comunes son el defacement y la redirección maliciosa.

Inyección SQL

La inyección SQL es un tipo de ciberataque que involucra declaraciones SQL maliciosas o códigos de aplicación que se inyectan en los campos de entrada del usuario. Este proceso permite a los atacantes obtener acceso al backend de la web o al contenido corrupto de la base de datos.

Ataque DDoS

Se trata de tráfico falso que proviene de equipos controlados por atacantes, a menudo llamados botnets. Un botnet es una red de computadoras hackeadas, conectadas a internet, que ejecutan uno o más bots. Si el ataque es lo suficientemente fuerte, el servidor puede bloquearse, desactivando así por completo la página web.
